## Overview

The `Reservas` component is the core booking interface where users select desks and time slots. It features an interactive desk map, real-time availability checking, and desk rotation enforcement.

**Route:** `/reservas`

## Component Purpose

- Interactive visual desk selection with map interface
- Real-time availability display for each desk
- Time slot selection (Morning, Afternoon, Full Day)
- Desk rotation enforcement (prevents booking same desk consecutively)
- Geofencing for booking time restrictions
- Admin override capabilities

## Date Generation

Generates next 2 business days (excludes weekends):

<CodeGroup>
```javascript Generate Business Days
const generarFechasHabiles = (n = 2) => {
  const fechas = [];
  const hoy = new Date();
  hoy.setHours(0, 0, 0, 0);
  let cursor = new Date(hoy);
  
  while (fechas.length < n) {
    const dow = cursor.getDay();
    if (dow !== 0 && dow !== 6) {  // Skip weekends
      fechas.push({
        date: new Date(cursor),
        iso: cursor.toISOString().split('T')[0],
        label: `${DIAS_CORTO[dow]}. ${dd}/${mm}`,
        chipLabel: formatoFecha,
      });
    }
    cursor.setDate(cursor.getDate() + 1);
  }
  return fechas;
};
```
</CodeGroup>

### Slot Blocking Logic

```javascript Calculate Blocked Slots
const turnosBloqueados = (reservas, puestoId) => {
  const bloq = new Set();
  reservas
    .filter(r => getPuestoId(r) === puestoId && esReservaActiva(r))
    .forEach(r => {
      const hId = getHorarioId(r);
      if (hId === H_AM)            { bloq.add(H_AM);  bloq.add(H_COMPLETO); }
      else if (hId === H_PM)       { bloq.add(H_PM);  bloq.add(H_COMPLETO); }
      else if (hId === H_COMPLETO) { bloq.add(H_AM);  bloq.add(H_PM); bloq.add(H_COMPLETO); }
    });
  return bloq;
};
```

<Note>
  If a desk has AM booked, both AM and Full Day are blocked. If Full Day is booked, all slots are blocked.
</Note>

## Rotation Enforcement

Prevents users from booking the same desk on consecutive booking days:

<CodeGroup>
```javascript Check Last Reservation
useEffect(() => {
  const documento = String(usuario?.document_number ?? '').trim();
  
  setLoadingRotacion(true);
  fetch(buildUserHistoryUrl(documento, fechaISO))
    .then(r => r.json())
    .then(json => {
      const data = Array.isArray(json.data) ? json.data : [];
      const ultimaActiva = data.find(esReservaActiva);
      const puestoId = getPuestoId(ultimaActiva);
      const fecha = ultimaActiva?.attributes?.fecha_reserva;
      setUltimaReservaPrevia(puestoId ? { puestoId, fecha } : null);
    })
    .finally(() => setLoadingRotacion(false));
}, [usuario?.document_number, fechaISO]);
```
</CodeGroup>

**Rotation Message:**
```javascript
const getRotationMessage = (puestoId) => {
  const fechaUltima = formatFechaIso(ultimaReservaPrevia?.fecha);
  return `Tu última reserva fue el ${fechaUltima} en el escritorio ${puestoId}. 
          Debes hacer rotación y elegir otro escritorio.`;
};
```

## Reservation Creation

<CodeGroup>
```javascript Create Reservation
const handleReservar = async (horarioObj) => {
  if (!usuario || !selectedId || !horarioObj) return;
  
  // Validation checks
  if (!canReserveNow) { setReservaErr(reserveWindowMessage); return; }
  if (loadingRotacion) { setReservaErr('Aún estamos validando...'); return; }
  if (yaReservoHoy) { setReservaErr('Ya tienes una reserva...'); return; }
  if (ultimoPuestoReservado && selectedId === ultimoPuestoReservado) {
    setReservaErr(getRotationMessage(selectedId));
    return;
  }
  
  setReservando(true);
  setReservaErr(null);
  
  try {
    const body = {
      data: {
        Nombre: usuario.nombre,
        foto: usuario.foto,
        documento: String(usuario.document_number),
        area: usuario.area_nombre,
        fecha_reserva: fechaISO,
        estado: null,  // Pending status
        working_puestos: { id: selectedId },
        working_horarios: { id: horarioObj.id },
      },
    };
    
    const res = await fetch(API_RESERVAS, {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify(body),
    });
    
    if (!res.ok) throw new Error(`Error ${res.status}`);
    
    setReservaOk(true);
    window.dispatchEvent(new CustomEvent('working-reservas-updated'));
    cargarReservas();
    
    setTimeout(() => {
      setSelectedId(null);
      setReservaOk(false);
    }, 2500);
  } catch (err) {
    setReservaErr(err.message || 'Error al reservar.');
  } finally {
    setReservando(false);
  }
};
```
</CodeGroup>
