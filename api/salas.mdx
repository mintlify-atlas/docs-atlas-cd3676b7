## Overview

The `Salas` component displays available coworking rooms and allows users to select one to proceed with their desk reservation.

**Route:** `/salas`

## Component Purpose

- Displays all available coworking rooms from Strapi CMS
- Shows room capacity and monitor availability
- Indicates room status (Available/Coming Soon)
- Allows admin users to access the admin panel

## API Integration

### Fetch Rooms Endpoint

```javascript
GET https://macfer.crepesywaffles.com/api/working-salas?populate=foto&sort=id:asc
```

**Response Structure:**
```json
{
  "data": [
    {
      "id": 1,
      "attributes": {
        "nombre": "Sala Principal",
        "capacidad": 6,
        "monitores": 3,
        "disponible": true,
        "foto": {
          "data": [
            {
              "attributes": {
                "url": "/uploads/sala1.jpg"
              }
            }
          ]
        }
      }
    }
  ]
}
```

## Data Normalization

The `normalizeSala` function transforms Strapi API responses into a consistent format:

<CodeGroup>
```javascript Normalize Function
const normalizeSala = (item) => {
  const a = item.attributes ?? item;

  // Handle image URLs from Strapi media library
  let imagenUrl = null;
  const fotoArr = a.foto?.data;
  if (Array.isArray(fotoArr) && fotoArr.length > 0) {
    const url = fotoArr[0].attributes?.url;
    if (url) imagenUrl = url.startsWith('http') ? url : `${BASE}${url}`;
  }

  return {
    id:         item.id,
    nombre:     a.nombre ?? a.name ?? `Sala ${item.id}`,
    capacidad:  a.capacidad ?? 0,
    monitores:  a.monitores ?? 0,
    imagen:     imagenUrl,
    disponible: a.disponible !== false,
    ruta:       '/reservas',
  };
};
```
</CodeGroup>

## Code Example

<CodeGroup>
```javascript Room Loading
const cargarSalas = useCallback(async () => {
  setLoading(true);
  setError(null);
  try {
    const response = await fetch(
      `${API_SALAS}?populate=foto&sort=id:asc`
    );
    const json = await response.json();
    const data = Array.isArray(json.data) ? json.data : [];
    setSalas(data.map(normalizeSala));
  } catch {
    setError('No se pudieron cargar las salas.');
  } finally {
    setLoading(false);
  }
}, []);
```
</CodeGroup>
