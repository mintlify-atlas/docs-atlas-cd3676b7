## Overview

The `Panel` component serves as both the user's reservation dashboard and the admin control panel. It displays reservations with filtering, search, and management capabilities including geolocation-based confirmation.

**Route:** `/panel`

## Component Purpose

- View and manage user reservations
- Admin panel for managing all reservations
- Geolocation-based reservation confirmation
- Real-time reservation sync
- Filter and search reservations
- Cancel, confirm, or reactivate reservations

## API Integration

### Reservations Endpoint

```javascript
GET https://macfer.crepesywaffles.com/api/working-reservas
```

**Query Parameters:**
- `sort[0]=fecha_reserva:desc`
- `sort[1]=id:desc`
- `filters[documento][$eq]={documento}` (non-admin)
- `filters[fecha_reserva][$eq]={date}` (today only)
- `populate[working_puestos]`
- `populate[working_horarios]`
- `pagination[pageSize]=40000`

## Geolocation-Based Confirmation

Users must be within workplace radius to confirm reservations:

<CodeGroup>
```javascript Geolocation Check
const refreshLocationStatus = async (silent = false) => {
  if (!geoSupport.supported) {
    setLocationError('Tu navegador no soporta geolocalización.');
    return;
  }

  setLocationChecking(true);
  
  try {
    const position = await getCurrentPosition();
    const distance = calculateDistance(
      position.latitude,
      position.longitude,
      workplaceInfo.latitude,
      workplaceInfo.longitude
    );

    setDistanceMeters(distance);
    setIsNearPoint(distance <= workplaceInfo.radius);
    setLocationError('');
  } catch (err) {
    setLocationError(err?.message || 'No fue posible obtener tu ubicación.');
    setIsNearPoint(false);
  } finally {
    setLocationChecking(false);
  }
};
```
</CodeGroup>

## Confirmation Time Window

Validates that confirmation happens within 25-minute window:

```javascript Time Validation
const getConfirmMeta = (reserva) => {
  const timeInfo = getVerificationTimeInfo(
    reserva,
    [],
    new Date(nowMs)
  );

  if (!timeInfo.isToday) {
    return {
      canByTime: false,
      blockedMessage: 'Solo puedes confirmar reservas del día de hoy.',
    };
  }

  if (!timeInfo.isActive) {
    return {
      canByTime: false,
      blockedMessage: timeInfo.message || 'La ventana de confirmación no está activa.',
    };
  }

  const remainingMinutes = timeInfo.verificationEnd
    ? Math.max(0, Math.ceil((timeInfo.verificationEnd.getTime() - nowMs) / 60000))
    : null;

  return {
    canByTime: true,
    remainingMinutes,
  };
};
```

## User Actions

### Confirm Reservation

<CodeGroup>
```javascript User Confirmation
const handleConfirmar = async (id) => {
  const reservaAux = reservations.find(r => r.id === id);
  
  // Check location requirement
  if (!isNearPoint) {
    alert(`Debes estar dentro de ${workplaceInfo.radius} metros...`);
    return;
  }

  setConfirmando(id);
  
  try {
    const evaluation = await verifyAttendance(reservaAux, {
      workplaceInfo,
      referenceDate: new Date(nowMs),
    });

    if (!evaluation.shouldUpdate) {
      alert(evaluation.message);
      return;
    }

    await updateReservaWithVerification(id, {
      estado: evaluation.newStatus,
      confirmada: evaluation.newStatus === 'Confirmada',
      verificacionAsistencia: {
        fecha: new Date().toISOString(),
        mensaje: evaluation.message,
        distancia: evaluation.distance,
        tipo: 'confirmacion-manual-geolocalizada',
      },
    }, reservaAux);

    // Update local state
    setReservations(prev => /* update logic */);
    alert(evaluation.message);
  } finally {
    setConfirmando(null);
  }
};
```
</CodeGroup>

### Cancel Reservation

```javascript Cancellation with Reason
const handleCancelar = async (id, motivoCancelacion) => {
  setCancelando(id);
  
  try {
    const reservaAux = reservations.find(r => r.id === id);
    await cancelReserva(id, reservaAux, motivoCancelacion);
    
    setReservations(prev =>
      prev.map(r => r.id === id ? {
        ...r,
        estado: 'Cancelada',
        confirmada: false,
        motivoCancelacion,
        verificacionAsistencia: {
          fecha: new Date().toISOString(),
          mensaje: motivoCancelacion,
          tipo: 'cancelacion-manual',
        },
      } : r)
    );
  } finally {
    setCancelando(null);
  }
};
```

### Reactivate (Admin Only)

```javascript Admin Reactivation
const handleReactivar = async (id) => {
  if (!esAdmin) return;

  setReactivando(id);
  
  try {
    await updateReservaWithVerification(id, {
      estado: 'Pendiente',
      confirmada: null,
      motivoCancelacion: null,
      verificacionAsistencia: {
        fecha: new Date().toISOString(),
        mensaje: 'Reserva reactivada por administrador.',
        tipo: 'reactivacion-admin',
        autorizadaPor: documentoUsuario,
      },
    }, reservaAux);

    alert('Reserva reactivada.');
  } finally {
    setReactivando(null);
  }
};
```

## Filtering & Search

### Combined Filter Logic

```javascript Filter Reservations
const filteredReservations = useMemo(() => {
  let result = reservations;
  
  // Admin tabs
  if (esAdmin) {
    if (adminTab === 'mis') result = result.filter(r => r.documento === documentoUsuario);
    else if (adminTab === 'otros') result = result.filter(r => r.documento !== documentoUsuario);
  }
  
  // Status filter
  if (filterEstado !== 'todos') {
    result = result.filter(r => r.estado === filterEstado);
  }
  
  // Room filter
  if (filterSala !== 'todas') {
    result = result.filter(r => String(r.salaId) === filterSala);
  }
  
  // Search query
  if (searchQuery.trim()) {
    const q = searchQuery.trim().toLowerCase();
    result = result.filter(r =>
      String(r.id).includes(q) ||
      (r.nombreCompleto ?? '').toLowerCase().includes(q) ||
      (r.documento ?? '').toLowerCase().includes(q) ||
      String(r.puestoId ?? '').includes(q)
    );
  }
  
  return result;
}, [reservations, esAdmin, adminTab, filterEstado, filterSala, searchQuery]);
```
