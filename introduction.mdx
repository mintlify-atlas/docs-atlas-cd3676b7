## Overview

Crepes & Waffles Coworking is a real-time coworking space reservation system designed for professionals and employees to book workstations with automated attendance verification. Built with React and integrated with real-time WebSocket synchronization, it provides a seamless experience for managing workspace reservations.

## How It Works

<Steps>
  <Step title="Employee Authentication">
    Users enter their employee ID number to authenticate and access the system
  </Step>
  <Step title="Select a Room">
    Browse available coworking rooms and view capacity, monitors, and availability
  </Step>
  <Step title="Choose Time Slot">
    Select from morning (8am-12pm), afternoon (1pm-5pm), or full day (8am-5pm)
  </Step>
  <Step title="Pick a Workstation">
    Choose an available desk from the interactive room layout
  </Step>
  <Step title="Verify Attendance">
    Check in using geolocation verification when you arrive at the workspace
  </Step>
</Steps>

## Technology Stack

The application is built with modern web technologies:

- **React 19** - UI framework with hooks and context
- **React Router** - Client-side routing
- **Ant Design** - UI component library
- **Vite** - Build tool and dev server
- **Axios** - HTTP client for API requests
- **WebSocket** - Real-time bidirectional communication
- **Tailwind CSS** - Utility-first CSS framework
- **Three.js** - 3D graphics for enhanced visualizations
- **Day.js** - Date manipulation library
