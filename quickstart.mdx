## Overview

Crepes & Waffles Coworking is a SaaS reservation system that enables employees to reserve workspaces with real-time synchronization, geolocation-based check-in, and automatic rotation policies.

**Key Features:**
- **Real-time Updates**: WebSocket sync keeps all users up-to-date instantly
- **Geolocation Check-in**: Confirm your reservation within the first 25 minutes using your device's location
- **Desk Rotation**: Automatic enforcement of desk rotation policies
- **Multi-shift Support**: Morning (AM), afternoon (PM), or full-day reservations

## Prerequisites

Before you begin, ensure you have:
- A valid employee ID (cédula)
- A device with location services enabled
- Modern web browser (Chrome, Firefox, Safari, or Edge)
- Access to the coworking space network

<Steps>

### Step 1: Validate Your ID

Start by entering your employee ID number (cédula) on the login screen.

```jsx src/assets/componentes/Validar_cedula.jsx
const handleUsuario = async (e) => {
  e.preventDefault();
  
  if (!cedula.trim()) {
    setError("Por favor ingrese el número de cédula");
    return;
  }
  
  setLoading(true);
  setError("");
  
  try {
    const response = await axios.get(
      `https://apialohav2.crepesywaffles.com/buk/empleados3?documento=${cedula}`
    );
    
    const empleado = response.data.data[0];
    const fechaHoraIngreso = new Date();
    
    navigate("/bienvenida", {
      state: {
        datosEmpleado: empleado,
        fechaHoraIngreso: fechaHoraIngreso.toISOString(),
      },
    });
  } catch (error) {
    console.error(error);
    setError("Error al validar la cédula.");
  } finally {
    setLoading(false);
  }
};
```

**What happens:**
1. Your ID is validated against the employee API
2. Your profile data (name, role, area, photo) is retrieved
3. You're automatically redirected to the welcome screen

### Step 5: Confirm Your Reservation

Review your selection and click "Reservar" to confirm.

```jsx src/assets/componentes/Reservas.jsx
const handleReservar = async (horarioObj) => {
  if (!usuario || !selectedId || !horarioObj) return;
  
  // Validation checks
  if (yaReservoHoy) {
    setReservaErr('Ya tienes una reserva para este día.');
    return;
  }
  
  if (ultimoPuestoReservado && selectedId === ultimoPuestoReservado) {
    setReservaErr(getRotationMessage(selectedId));
    return;
  }
  
  const body = {
    data: {
      Nombre:           usuario.nombre ?? '',
      foto:             usuario.foto ?? '',
      documento:        String(usuario.document_number),
      area:             usuario.area_nombre ?? '',
      fecha_reserva:    fechaISO,
      estado:           null,  // Pending
      working_puestos:  { id: selectedId },
      working_horarios: { id: horarioObj.id },
    },
  };
  
  const res = await fetch(API_RESERVAS, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(body),
  });
  
  // Trigger real-time sync
  window.dispatchEvent(new CustomEvent('working-reservas-updated'));
};
```

**What gets stored:**
- Your employee information (name, ID, photo, area)
- Reservation date
- Desk ID and time slot
- Initial status: `Pending`

### Step 6: Check In with Geolocation

On the day of your reservation, confirm your attendance within the first 25 minutes of your shift.

```jsx src/assets/componentes/VerificacionAsistencia.jsx
const verify = useCallback(async () => {
  // Check geolocation support
  if (!geoSupport.supported) {
    return {
      success: false,
      message: 'Tu navegador no soporta geolocalización.'
    };
  }
  
  // Request location permission
  const hasPermission = await requestLocationPermission();
  if (!hasPermission) {
    return {
      success: false,
      message: 'Debes habilitar la ubicación para confirmar la reserva.'
    };
  }
  
  // Verify attendance
  const localVerification = await verifyAttendance(reserva, { workplaceInfo });
  
  if (!localVerification.success) {
    return {
      success: false,
      message: localVerification.message
    };
  }
  
  // Submit verification to API
  const apiResult = await postVerificarAsistencia({
    reservaId,
    lat: position.latitude,
    lng: position.longitude,
  });
  
  return {
    success: true,
    message: 'Reserva confirmada por geolocalización.'
  };
}, [reserva, workplaceInfo]);
```

**Check-in Requirements:**
1. **Time Window**: Within 25 minutes of shift start
2. **Location**: Within 1000m radius of coworking space
3. **Browser Permission**: Location services enabled

<Warning>
If you don't check in within 25 minutes, your reservation will be automatically cancelled to free up the desk for others.
</Warning>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Location services not available">
    If your device doesn't support location services or you can't enable them:
    1. Visit reception desk
    2. Provide your reservation ID
    3. Staff will manually confirm your check-in
  </Accordion>
  
  <Accordion title="Cannot select a desk">
    This usually means:
    - The desk is fully booked for your selected time
    - You reserved this desk yesterday (rotation policy)
    - You already have an active reservation for today
  </Accordion>
  
  <Accordion title="Reservation automatically cancelled">
    Reservations are auto-cancelled if:
    - Not confirmed within 25 minutes of shift start
    - You're outside the 1000m radius during check-in
    - You attempt to violate rotation policies
  </Accordion>
</AccordionGroup>
